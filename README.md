# 🖥️ LG Mini Browser for webOS

A lightweight **custom web browser** built for **LG Smart TVs (webOS)**.  
This project provides a minimal yet functional browsing experience optimized for TV screens — perfect for local dashboards, web apps, or custom kiosks.

---

## 🚀 Features

- 🌐 Full-screen browsing via iframe
- ⚙️ URL input dialog with quick access
- 🧭 Persistent URL storage (remembers last site)
- 🧱 Built-in **error screen** for unreachable pages
- 📶 **Loading progress bar** with smooth animation
- 🎨 Modern, minimalist design with simple UI
- 🕹️ Supports TV remote navigation (via webOS APIs)
- 💾 Packaged as `.ipk` for easy installation on LG TVs

---

## 📦 Installation on LG TV

### Requirements
- LG Smart TV running **webOS 3.0+**
- **Developer Mode** enabled on your TV  
  (Enable via *LG Developer Mode* app from LG Content Store)
- [**webOS CLI Tools**](https://webostv.developer.lge.com/sdk/tools/install-instructions) installed on your computer
- PC and TV connected to the **same Wi-Fi network**
